# BAGIAN 1 — SINGLE SOURCE OF TRUTH

## MASALAH

Saat ini AI, Flight Cards, dan Route DB masih dapat menghasilkan informasi berbeda.

Contoh:

AI:
Harga $800-$1200

Card:
$614

Route DB:
$180

Ini tidak boleh terjadi.

## WAJIB

Semua komponen wajib membaca object yang sama.

```ts
interface ActiveSearch {
  requestId: string;

  origin: string;
  destination: string;
  date: string;

  flights: Flight[];

  lowestPrice: number;
  highestPrice: number;

  airlines: string[];

  source: "live" | "cache" | "fallback";

  timestamp: number;
}
```

Flow wajib:

Search Engine
↓
ActiveSearch
↓
AI Summary
↓
Flight Cards

Tidak boleh ada source lain.
