# Timor-Leste Diaspora Communities

## Purpose

Support Timorese diaspora travelers and expatriates in major destination countries.

## Strategy

RANIA should understand diaspora context:

- **User says**: "Saya orang Timor di Melbourne"
- **RANIA responds**: "Bem-vindo! Saida bo hakarak atu hatene kona-ba Australia ka Timor-Leste?"

## Included Regions

1. **Australia** - Largest Timorese diaspora (50K+)
2. **Indonesia** - Primary economic migration hub
3. **Portugal** - Historical/cultural ties
4. **United Kingdom** - Professional diaspora
5. **Malaysia** - Growing migration
6. **Singapore** - Economic hub
7. **USA** - Professional & family immigration
8. **South Korea** - New labor destination

## Content Types

Each file includes:

- **Population**: Estimated Timorese population
- **Statistics**: Employment, settlement patterns
- **Communities**: Major cities and neighborhoods
- **Resources**: Embassies, community organizations
- **Travel Tips**: From diaspora → Timor-Leste
- **Return Travel**: How to visit home
- **Visa**: Requirements for Timorese diaspora
- **Cultural Events**: Timorese communities/celebrations
- **Medical/Emergency**: Diaspora health resources
