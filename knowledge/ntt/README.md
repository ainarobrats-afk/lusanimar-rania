# NTT Knowledge (Nusa Tenggara Timur)

Panduan singkat dan roadmap untuk konten NTT: budaya, pariwisata, kuliner, transport, kota, dan daerah perbatasan. Prioritas: data yang sering ditanyakan pengguna dari/ke Timor-Leste.

Folder contents:
- `culture.md` — ringkasan adat & region
- `tourism.md` — tempat wisata utama
- `food.md` — makanan khas
- `transport.md` — rute + akses
- `cities.md` — profil kota-kota penting
- `border_areas.md` — titik perbatasan relevan

Format: gunakan markdown ringkas; setiap file harus mudah di-parse oleh worker runtime.
